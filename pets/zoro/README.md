# Zoro Codex Pet

Fan-made Zoro-inspired Codex pet with sword idle animations, dash attacks, jumping sword combos, lost-direction states, review focus, and failed-task reactions.

## Files

- `pet.json`
- `spritesheet.webp`

## Install

From the repo root, use the main installer:

```powershell
irm https://raw.githubusercontent.com/anisayari/codex-pets/main/install.ps1 | iex
```

Then reload Codex with `Ctrl+K` -> `Force Reload Skills`, or restart Codex, and select `Zoro` in `Settings` -> `Appearance` -> `Pets`.

This is an unofficial fan-made pet and is not affiliated with or endorsed by the original rights holders.
